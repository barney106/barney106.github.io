# Molino Apps - Sitios Web de Soporte

Este repositorio contiene la landing page y las políticas de privacidad para las aplicaciones Android desarrolladas por **Molino Apps**.

## Contenido

- **Landing Page**: Página principal con el catálogo de aplicaciones.
- **Políticas de Privacidad**: Documentación requerida por el programa de desarrolladores de Google Play.
- **app-ads.txt**: Archivo de autorización para publicidad digital.

## Despliegue

El sitio se despliega automáticamente a través de GitHub Pages.
